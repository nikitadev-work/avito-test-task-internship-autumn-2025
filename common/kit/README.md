## Документация для common/kit

`common/kit` — это вспомогательный модуль, содержащий переиспользуемые инфраструктурные компоненты.  
Модуль оформлен как отдельный Go-модуль (`go.mod` внутри каталога), что позволяет подключать его из разных сервисов проекта.



## Основные компоненты

### 1. Логгер (`common/kit/logger`)

Функциональность:
- структурированное логирование в формате JSON;
- уровни логов (`debug`, `info`, `warn`, `error`);
- добавление контекстных полей (`service`, `version`, технические параметры);
- реализует интерфейс `LoggerInterface`, используемый сервисом.

---

### 2. Метрики (`common/kit/metrics`)

Функциональность:
- бизнес-метрики:
  - количество созданных команд;
  - активированные/деактивированные пользователи;
  - созданные/смёрженные PR;
  - количество переназначений ревьюеров;
- интеграция с Prometheus через эндпоинт `/metrics`;
- полная совместимость с Grafana.

## Назначение

Модуль создан для:
- унифицированного логирования;
- унифицированного сбора метрик;
- уменьшения дублирования кода между сервисами;
- повышения читаемости и поддержки проекта.
